# Venture Builder

A modern Venture Builder website showcasing equity partnerships and software development services.

## Portfolio

- **Visualiza.ai** - AI-powered audience analysis
- **Fluxxpass** - Digital pass management platform
- **Synapsis** - AI assistant platform

## Services

- Equity Partnerships for startups
- Software Development for businesses

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
